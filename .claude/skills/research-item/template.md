## Notes on File Structure

- **Filename:** `fires/YYYY-MM-DD-{city-state}-{short-description}.md`
  - Use the date the fire *started* (not reported or arrested).
  - If date is uncertain, use the earliest plausible date with a note.
- **Status values:** Confirmed arson / Suspected arson / Under investigation / Accidental / Not arson / Unconfirmed
- **Labor connection values:** Explicit / Indirect / None / Unknown
- **Sections are flexible** — omit sections that don't apply (e.g., no Suspect section if no arrests), add sections if warranted (e.g., "Injuries" as a standalone section if significant).
- **Quotes in body text** should be linked to the source article.
- **Every source cited twice:** once as a hyperlink in the body (when quoting or referencing), and once in the Sources list at the end.
